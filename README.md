# ZYNCO Online Shopping Mall

This is a premium, responsive e-commerce website for phone accessories built with HTML, CSS (Tailwind via CDN), and JavaScript.

## How to Run
Simply double-click `index.html` to open the website in your browser. No installation or Node.js required.

## Features
- **Modern Dark UI**: Glassmorphism effects, smooth animations, and a premium color palette.
- **Shop & Filter**: Browse products with category and price filters.
- **Shopping Cart**: Fully functional cart that saves your items even if you refresh the page (uses LocalStorage).
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **No Dependencies**: Uses CDN links for Tailwind and FontAwesome, so it's lightweight.

## Customization
- **Products**: Edit `js/products.js` to add or change products.
- **Colors**: You can modify the Tailwind config in the `<head>` of each HTML file.

Enjoy your new website!
